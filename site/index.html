Everything in one place

Stop switching between terminal, browser, and design tools.

💬

Chat input

Real text box with Enter to send. No more pasting prompts into a raw terminal.

🌐

Browser preview

See localhost side-by-side. Auto-detects dev servers. Desktop, tablet, and phone sizes.

🎨

Design inspector

Click elements to inspect. Tweak styles live. Send changes to your agent as instructions.

📑

Sessions

Named tabs grouped by project. Switch with Cmd+1-9. Persist across restarts.

🧠

Context aware

Each agent knows about sibling sessions and your tools. Smarter suggestions.

🎭

10+ themes

Full visual editor. Share themes via URL. Every pixel is customizable.

📋
Paste images
Cmd+V screenshots directly into the chat. They get sent to the agent.
⌨️
Keyboard-first
Full shortcut set. Cmd+B browser, Cmd+D inspect, Cmd+1-9 tabs.
📱
Device frames
Preview at desktop, tablet, or phone width with one click.
🔗
Auto URL detection
Agent starts a dev server? Deck catches the URL and opens it.
🤖
AI session naming
Sessions auto-name themselves based on what you're working on.
💾
Persistent state
Quit and relaunch. Sessions, tabs, groups — all restored.
